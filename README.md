# Paws and Preferences

A React + TypeScript + Vite project.

## Features
- React 18 frontend
- TypeScript for type safety
- Vite for fast development and Hot Module Replacement (HMR)
- ESLint configured for TypeScript and React

## Prerequisites
- Node.js v18+ installed
- npm or yarn package manager

## Setup

1. **Clone the repository**
```bash
git clone https://github.com/DennisNg-03/paws-and-preferences.git
cd paws-and-preferences
```

2. **Install dependencies**
```
npm install
```

3. **Start development server**
```
npm run dev
```
Open `http://localhost:5173` in your browser to access the web application.


**(Optional) Preview production build locally**
```
npm run preview
```

**(Optional) Build for production**
```
npm run build
```
